# Birko.Data.Migrations.RavenDB

RavenDB migration provider for the Birko Framework.

## Features

- Index management migrations
- Bulk operations and query-based deletions
- RavenDBMigration base class

## Installation

```bash
dotnet add package Birko.Data.Migrations.RavenDB
```

## Dependencies

- Birko.Data.Core (AbstractModel)
- Birko.Data.Stores (Settings)
- Birko.Data.Migrations
- Birko.Data.RavenDB
